# Health Engine

Health Engine is a recovery and readiness dashboard built on physiological trends,
feature-engineered signals, and a trained recovery prediction model.

## What this Space includes

- Manual readiness dashboard with live score updates
- Recovery prediction model backed by a trained random forest
- Historical trend views for HRV, sleep, resting heart rate, strain, and blood oxygen
- Batch CSV scoring workflow

## Runtime notes

This Space is packaged from the local project with a minimal dependency set and the
latest trained model artifact.
